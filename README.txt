Software Dependencies
1. C compiler such as gcc or icc
2. FFTW library

Compilation
1. Change folder to 'src'
2. Compile by typing 'make' at the commandline. Edit 'Makefile' as appropriate.

Reconstruction demo
1. Change folder to 'demo'
2. Run the script 'run.sh' at the commandline. Edit 'run.sh' as appropriate.
